📡 AmpliFi Alien Exporter

A Prometheus exporter for AmpliFi Alien routers. Exposes detailed WiFi client metrics, router info, ethernet port stats, and more — all scraped directly from the router's local API.

🚀 Quick Start

docker run -d \
  -e AMPLIFI_PASSWORD=your_router_password \
  -e AMPLIFI_ROUTER_IP=192.168.1.1 \
  -p 9877:9877 \
  ghcr.io/matanbaruch/amplifi-alien-exporter:latest

⚙️ Environment Variables

VariableDefaultRequiredDescription
AMPLIFI_ROUTER_IP192.168.1.1NoRouter IP address
AMPLIFI_PASSWORD—YesRouter admin password
SCRAPE_INTERVAL15NoScrape interval in seconds
LISTEN_PORT9877NoHTTP port to expose metrics on
LISTEN_ADDR0.0.0.0NoListen address
